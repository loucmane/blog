## Memory Naming Reform
Moving forward, memories should follow:
- Format: `[yyyymmdd]-[type]-[brief-topic]`
- Types: session, feature, lesson, fix, review
- Example: `20250108-session-work-tracking`

## Connection Strategy
- Work folders reference memories

- SESSION.md links to both
- Enables full context reconstruction
