## Decisions Made

2. **Monthly organization**: Completed work organized by year/month
3. **Status in folder name**: -ACTIVE, -DONE, -BLOCKED
4. **Phase/sprint prefix**: For better context
5. **Memory refs file**: Separate file for memory connections
