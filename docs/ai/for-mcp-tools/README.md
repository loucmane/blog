# MCP Tools Documentation

## Overview
This directory contains documentation for various MCP (Model Context Protocol) tools that enhance the development workflow. This includes specialized tools beyond the primary ones covered in other directories.

## Directory Structure
- **prompts/**: Pre-written prompts for MCP tool operations
- **examples/**: Real-world usage examples
- **guidelines/**: Best practices for MCP tool usage
- **reference/**: Technical reference for each tool

## Common MCP Tools
- **IDE Integration**: `mcp__ide__getDiagnostics`, `mcp__ide__executeCode`
- **Context7**: `mcp__context7__resolve-library-id`, `mcp__context7__get-library-docs`
- **Server Status**: Various status and monitoring tools
- **Other specialized tools**: As they become available

## Integration with TWES
MCP tools are fundamental to the Total Workflow Excellence System, providing:
- Real-time diagnostics and feedback
- Documentation retrieval and context
- Code execution capabilities
- Integration with external services

## Best Practices
- Always check tool availability with appropriate commands
- Use tools in combination for complex workflows
- Monitor tool performance and availability
- Document new tool discoveries
